# Image Manifest

All images live on Framer's public CDN. Base URL pattern:
`https://framerusercontent.com/images/<FILENAME>`
(append `?scale-down-to=1024` for a smaller version).

> Note on downloading: these are binary files on a CDN. They can be referenced by
> URL directly, or downloaded locally with the script `download-images.ps1` in the
> project root (run it in PowerShell to populate an `assets/images/` folder).
